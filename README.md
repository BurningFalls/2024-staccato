# ✈️ Staccato
> `스타카토`는 '**여행의 추억을 지도 위에 함께 기록하고 공유하는 서비스**'입니다.<br>

`스타카토` 는 단조로운 선율에 변화를 주거나 특정 부분을 강조합니다. 단조로운 일상이라는 악보에 여행이라는 스타카토를 찍어보아요!
